List - GRUPO 06

TEORÍA
¿Qué es un List Group?

El componente List Group de Bootstrap 5 permite crear listas estilizadas y bien estructuradas de elementos. Es un componente muy versátil que se puede utilizar para menús de navegación, listas de tareas, listas de productos, o cualquier grupo de contenido relacionado que deba presentarse de forma organizada.

Importancia de los Grupos de Listas

Los grupos de listas son una herramienta flexible y potente para mostrar una serie de elementos relacionados. Permiten al usuario visualizar información estructurada de manera clara y ordenada, mejorando la accesibilidad y la experiencia de navegación.

Ejemplo Práctico

Un ejemplo común del uso de los List Groups es en los menús de navegación de una página web. Dichos menús están formados por un conjunto de palabras o enlaces que, al actuar en conjunto, constituyen una estructura coherente. En este caso, el significado y la utilidad del grupo es mayor que la de cada elemento por separado.

Listas en HTML

En HTML, existen distintos tipos de listas para agrupar varios elementos de forma lógica. Estas listas pueden representarse de manera ordenada, no ordenada o definida, dependiendo de la información que se desea mostrar.

Tipos de Listas Principales:
  • Lista No Ordenada (<ul>): Agrupa elementos sin un orden específico, representados con viñetas.
  • Lista Ordenada (<ol>): Presenta los elementos en un orden numérico o alfabético.
  • Lista de Definición (<dl>): Se usa para mostrar pares de término y descripción, similar a un glosario.
Características Principales del List Group:
  • Estilización Moderna: Cada elemento de la lista tiene un diseño atractivo y uniforme.
  • Interactividad: Puede incluir enlaces, botones y elementos activos o deshabilitados.
  • Compatibilidad: Funciona con cualquier tipo de contenido: texto, imágenes o íconos.
  • Responsive: Se adapta automáticamente a diferentes tamaños de pantalla.
  • Integración: Puede combinarse con otros componentes como cards, tabs o modals.

Explicación Detallada de las Listas en Bootstrap 5
1. Listas No Ordenadas

Se crean con la etiqueta <ul> y cada elemento se define con <li>. Son útiles para mostrar conjuntos sin orden específico. Bootstrap permite personalizarlas fácilmente mediante clases como list-unstyled o list-inline. 


<ul class="list-unstyled">
  <li>Inicio</li>
  <li>Noticias</li>
  <li>Artículos</li>
  <li>Contactos</li>
</ul>
2. Listas Ordenadas

A diferencia de las anteriores, las listas ordenadas usan <ol> y numeran automáticamente los elementos. Son ideales para pasos secuenciales o jerárquicos. 


<ol>
  <li>Requerimientos</li>
  <li>Diseño</li>
  <li>Implementación</li>
  <li>Verificación</li>
  <li>Mantenimiento</li>
</ol>
3. List-Group Básica

El componente list-group de Bootstrap permite mostrar elementos con un estilo limpio y uniforme. Cada ítem se define con list-group-item. También puedes añadir active para marcar un elemento seleccionado. 


<ul class="list-group">
  <li class="list-group-item">Imágenes</li>
  <li class="list-group-item">Archivos</li>
  <li class="list-group-item">Audios</li>
  <li class="list-group-item">Videos</li>
  <li class="list-group-item">Mensajes</li>
</ul>

EJERCICIO

En esta sección encontrarás un ejercicio práctico para implementar las Listas. Incluye la explicación detallada del código principal con imágenes ilustrativas y la ejecución del código directamente en la página web.

Explicación del Código Principal
1. Creacion de listas

Como podemos ver en el siguiente ejercicio, las listas se pueden usar para crear menus, e incluso una barra de navegación; ya sea usando las etiquetas <ul> o <div>.

Con la clase list-group damos un formato visualmente atractivo y uniforme, agrupando los elementos; y con list-group-item nos encargamos de cada elemento de forma individual. 

<ul class="dropdown-menu list-group">
    <li class="list-group-item"><a href="#">Grupo 1</a></li>
    <li class="list-group-item"><a href="#">Grupo 2</a></li>
    <li class="list-group-item"><a href="#">Grupo 3</a></li>
    <li class="list-group-item"><a href="#">Grupo 4</a></li>
    <li role="separator" class="divider"></li>
    <li class="list-group-item"><a href="#">Grupo 5</a></li>
    <li class="list-group-item active"><a href="#">Grupo 6</a></li>
    <li class="list-group-item"><a href="#">Grupo 7</a></li>
    <li class="list-group-item"><a href="Tabla.html">Grupo 8</a></li>
</ul>
2. Lista de grupos

A cada elemento de la lista debemos darle la clase list-group-item para que pueda tener el formato, caso contrario se vera aislado de este cambio. Asi mismo, se pueden aplicar otras clases para mejorar la visualizacion de la lista como es active que muestra al elemento como seleccionado. 

<li class="list-group-item active"><a href="#">Grupo 6</a></li>
3. Variantes

Las clases contextuales dan un estilo al enlace o elemento de la lista, sin la necesidad de usar CSS, bien podemos usar success, info, warning, entre otros. En otras palabras, es como dar color de fondo al elemento de la lista.

Incluso podemos usar iconos fuente para resaltar la idea que debe dar el elemento. 

<div class="row list-group" id="cuerpo">
    <!-- Left menu -->
    <aside id="leftMenu" class="col-xs-12 col-sm-4 col-md-3 col-lg-2">
        <a href="FormLogin.html" class="btn btn-default btn-block list-group-item list-group-item-success">
            <span class="glyphicon glyphicon-user"></span> Login
        </a>
        <a href="FormRegistro.html" class="btn btn-default btn-block list-group-item list-group-item-info">
            <span class="glyphicon glyphicon-hdd"></span> Registro
        </a>
        <a href="FormProducto.html" class="btn btn-default btn-block list-group-item list-group-item-warning">
            <span class="glyphicon glyphicon-book"></span> Producto
        </a>
        <a href="#" class="btn btn-default btn-block list-group-item list-group-item-danger" data-toggle="modal" data-target="#miModal">
            <span class="glyphicon glyphicon-user"></span> Formulario
        </a>
    </aside>
</div>

En este caso, se usan como notificaciones para conocer la cantidad. Se usa la etiqueta <span> junto con la clase code badge para crear esta notificacion. Asi, el usuario puede identificar informacion rapidamente.

Esto tambien sirve para agregar ciertos elementos al footer de una pagina web. En este caso, podemos su aplicacion para representar ciertos temas, acompañados de iconos que los represetan.
