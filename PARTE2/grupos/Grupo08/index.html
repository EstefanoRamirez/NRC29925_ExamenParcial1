1. TEORÍA: ¿Qué es un Badge?

Los **Badges (insignias)** son pequeños componentes de contador o etiquetado que se utilizan para mostrar información adicional, como el número de notificaciones, mensajes no leídos o el estado de un elemento.

2. EJEMPLO: Sintaxis y Uso de Badges
2.1 Badges Contextuales
<span class="badge bg-primary">Primary</span>
Resultado: Primary
2.2 Badges tipo "Pill" (Píldora)
<span class="badge rounded-pill bg-success">Success</span>
Resultado: Success
2.3 Badges dentro de Botones
<button type="button" class="btn btn-dark position-relative">...</button>
Resultado:
1. TEORÍA: ¿Qué es un Breadcrumb?

Un **Breadcrumb (migas de pan)** es un componente de navegación secundario que indica la ubicación del usuario dentro de la jerarquía de un sitio web.

2. EJEMPLO: Sintaxis de Breadcrumbs
2.1 Breadcrumb Básico
<ol class="breadcrumb">
    <li class="breadcrumb-item"><a href="#">Inicio</a></li>
    <li class="breadcrumb-item active" aria-current="page">Datos</li>
</ol>
Resultado:
  1. Inicio
  2. Datos
2.2 Breadcrumb con Iconos
<ol class="breadcrumb">
    <li class="breadcrumb-item"><a href="#"><i class="fas fa-home"></i> Inicio</a></li>
    <li class="breadcrumb-item active" ...><i class="fas fa-book"></i> Biblioteca</li>
</ol>
Resultado:
  1. Inicio
  2. Biblioteca
1. TEORÍA: ¿Qué es Pagination?

El componente **Pagination (Paginación)** se utiliza para dividir contenido extenso en varias páginas, permitiendo al usuario navegar entre ellas.

2. EJEMPLO: Sintaxis de Pagination
2.1 Paginación Básica
<ul class="pagination">
    <li class="page-item"><a class="page-link" href="#">1</a></li>
    <li class="page-item active"><span class="page-link">2</span></li>
    <li class="page-item"><a class="page-link" href="#">3</a></li>
</ul>
Resultado:
  • 1
  • 2
  • 3
2.2 Paginación con Iconos
<ul class="pagination">
    <li class="page-item"><a class="page-link" href="#">«</a></li>
    <li class="page-item"><a class="page-link" href="#">»</a></li>
</ul>
Resultado:
  • «
  • »
2.3 Estados (Activo y Deshabilitado)
<ul class="pagination">
  <li class="page-item disabled"><span class="page-link">Anterior</span></li>
  <li class="page-item active"><span class="page-link">1</span></li>
</ul>
Resultado:
  • Anterior
  • 1
2.4 Tamaños (Sizing)
<ul class="pagination pagination-lg">...</ul>
<ul class="pagination pagination-sm">...</ul>
Resultado:
  • 1
  • 1
1. TEORÍA: ¿Qué es un Label?

El componente **Label (Etiqueta)** era usado en versiones anteriores de Bootstrap (como Bootstrap 3). En **Bootstrap 5**, esta funcionalidad se ha fusionado con el componente .badge.

2. EJEMPLO: Sintaxis de Labels (usando Badges)
2.1 "Labels" Básicos

Para crear una etiqueta, simplemente usa la clase .badge.

<h3>Título <span class="badge bg-secondary">Nuevo</span></h3>
Resultado:

Título de Ejemplo Nuevo

2.2 "Labels" de Colores
<span class="badge bg-success">Completado</span>
<span class="badge bg-warning text-dark">Pendiente</span>
Resultado: Completado Pendiente

Este es el ejercicio de laprueba. Se le han añadido las funcionalidades (como Badges/Labels para contadores y estados) que vimos en esta página, demostrando su aplicación en un entorno real.

Descripción del Ejercicio

Este es un proyecto web completo, basado en los temas tratados.

  • Páginas: Incluye index.html, Formulario.html, Comandante.html y Grupo10.html.
  • Estilos: Usa sus propias hojas de estilo en css/ y Recursos/.
  • Fuentes: Carga sus propias fuentes de iconos (fonts/glyphicons... y fonts/icomoon...).

Se puede navegar por este mini-sitio (el iframe de abajo) como si fuera una página independiente, pero sin salir de esta sección.

Explicación del Código Principal

En nuestro proyecto web, implementamos los nuevos componentes (Breadcrumbs, Pagination, y Labels/Badges) en varios lugares clave para mejorar la navegación y la interfaz. A continuación, se detalla cada implementación.

1. Labels de Contador en el Menú (index.html)

Se añadieron contadores al menú de la izquierda. En Bootstrap 3 (que usa el proyecto), se utiliza .label con clases de color (ej. .label-danger) y .pull-right para crear "badges" coloridos que indican notificaciones.

<ul class="nav nav-pills nav-stacked">
  <li>
    <a href="FormRegistro.html">
        <span class="label label-danger pull-right">2</span> 
        <span class="glyphicon glyphicon-hdd"></span> Registro
    </a>
  </li>
  <li>
    <a href="FormProducto.html">
        <span class="label label-primary pull-right">5</span>
        <span class="glyphicon glyphicon-book"></span> Producto
    </a>
  </li>
  <li>
    <a href="Formulario.html">
        <span class="label label-success pull-right">1</span>
        <span class="glyphicon glyphicon-list-alt"></span> Formulario
    </a>
  </li>
</ul>
