Tooltip and Popover - Grupo 01


¿Qué es Tooltip?

Un Tooltip es un pequeño cuadro de texto que aparece cuando el usuario pasa el cursor sobre un elemento de la interfaz gráfica de usuario (GUI). Su propósito principal es proporcionar información adicional, una breve explicación o el nombre de un elemento sin ocupar espacio permanente en la pantalla. Son una forma eficaz de ofrecer ayuda contextual sin abrumar al usuario.

¿Cuándo conviene usar un Tooltip?

  • Para aclarar iconos o botones sin texto.
  • En formularios o campos de entrada, para guiar al usuario.
  • Al mostrar abreviaturas o términos técnicos, explicando su significado.
  • En elementos deshabilitados o inactivos, cuando se necesita dar contexto o motivo.

Características clave


Brevedad
La información debe ser concisa y al grano. No están diseñados para párrafos largos.
Contextual
La información que proporcionan es relevante para el elemento al que están asociados.
Temporal
Aparecen al interactuar (al pasar el ratón) y desaparecen cuando la interacción termina (al quitar el ratón).
No interactivo
Los tooltips generalmente no son clicables ni permiten la interacción directa con el texto que contienen.

Ejemplo de Tooltip con Bootstrap 5


Para crear un tooltip, agregue el data-bs-toggle="tooltip" como atributo a un elemento. Utilice el title como atributo para especificar el texto que debe mostrarse dentro del tooltip.

HTML
<button class="btn btn-info btn-block" data-bs-toggle="tooltip" data-bs-placement="left" title="Hola">
    Presionar
</button>

Nota: Los tooltips no se activan solo con los atributos HTML, deben inicializarse con JavaScript para funcionar en todo el documento.

El siguiente fragmento de código permite habilitar todos los tooltips presentes en la página, buscando los elementos que contienen el atributo data-bs-toggle="tooltip" y activándolos de forma automática.

JavaScript

var tooltipTriggerList = [].slice.call(document.querySelectorAll('[data-bs-toggle="tooltip"]'))
var tooltipList = tooltipTriggerList.map(function (tooltipTriggerEl) {
  return new bootstrap.Tooltip(tooltipTriggerEl)
})

Tooltips de posicionamiento


Por defecto, el tooltip aparecerá encima del elemento. Al utilizar el atributo data-bs-placement se puede establecer la posición de la información sobre herramientas en la parte superior, inferior, izquierda o derecha del elemento

  • data-bs-placement="right" muestra el tooltip a la derecha del botón.
  • data-bs-placement="left" muestra el tooltip a la izquierda del botón.
  • data-bs-placement="top" muestra el tooltip en la parte superior del botón.
  • data-bs-placement="bottom" muestra el tooltip en la parte inferior del botón.

Ejemplo Tooltip

<div class="row">
    <div class="col-xs-12 col-sm-3">
    <button class="btn btn-info btn-block" data-bs-toggle="tooltip" data-bs-placement="right" title="Derecha">Presionar 1</button>
    </div>
    <div class="col-xs-12 col-sm-3">
    <button class="btn btn-info btn-block" data-bs-toggle="tooltip" data-bs-placement="left" title="Izquierda">Presionar 2</button>
    </div>
    <div class="col-xs-12 col-sm-3">
    <button class="btn btn-info btn-block" data-bs-toggle="tooltip" data-bs-placement="top"  title="Arriba">Presionar 3</button>
    </div>
    <div class="col-xs-12 col-sm-3">
    <button class="btn btn-info btn-block" data-bs-toggle="tooltip" data-bs-placement="bottom" title="Abajo">Presionar 4</button>
    </div>
</div>

¿Qué es Popover?

Un Popover es un pequeño cuadro superpuesto que aparece cuando el usuario interactúa con un elemento de la interfaz de usuario (haciendo clic). A diferencia de un Tooltip, un Popover está diseñado para mostrar contenido más rico y complejo a menudo interactivo y puede contener detalles contextuales, opciones o formularios pequeños , conservando la fluidez de la interacción.

¿Cuándo conviene usar un Popover?

  • Cuando se necesita mostrar más información que en un tooltip
  • En formularios o ayuda contextual
  • Para mostrar contenido adicional bajo demanda
  • Cuando se desea mantener al usuario en el mismo contexto visual

Características clave


Interactivo
A menudo permite al usuario interactuar con el contenido dentro del Popover (hacer clic en enlaces, rellenar campos, etc.).
Persistentemente
Permanece visible hasta que el usuario hace clic fuera de él, presiona Esc, o interactúa de otra manera para cerrarlo, no solo al quitar el cursor.
Contenido rico
Puede albergar mucho más que texto plano, incluyendo imágenes, enlaces, botones e incluso pequeños formularios.
Tiene un "puntero"
A menudo incluyen una pequeña flecha que apunta al elemento que lo activó, reforzando la relación contextual.

Ejemplos de Popover con Bootstrap 5


Para crear un popover, agregue el data-bs-toggle="popover" como atributo a un elemento. Además, utilice el title atributo para especificar el texto del encabezado del popover y data-bs-content atributo para especificar el texto que debe mostrarse dentro del cuerpo del popover:

HTML

<button class="btn btn-info" data-bs-container="body" data-bs-toggle="popover" data-bs-placement="right" data-bs-content="Lorem ipsum dolor sit amet, consectetur adipiscing elit. Sed pharetra consequat felis, ac efficitur mauris vestibulum non." title="Título Mensaje 1">
    Presionar
</button>

Nota: Los tooltips no se activan solo con los atributos HTML, deben inicializarse con JavaScript para funcionar en todo el documento.

El siguiente fragmento de código permite habilitar todos los popovers presentes en la página, buscando los elementos que contienen el atributo data-bs-toggle="popover" y activándolos de forma automática.

var popoverTriggerList = [].slice.call(document.querySelectorAll('[data-bs-toggle="popover"]'))
var popoverList = popoverTriggerList.map(function (popoverTriggerEl) {
  return new bootstrap.Popover(popoverTriggerEl)
})

Popover de posicionamiento

Por defecto, el popover aparecerá al lado derecho del elemento. Al utilizar el atributo data-bs-placement como atributo para establecer la posición del popover en la parte superior, inferior, izquierda o derecha del elemento

  • data-bs-toggle="popover" data-bs-placement="right" muestra el popover a la derecha del botón.
  • data-bs-toggle="popover" data-bs-placement="left" muestra el popover a la izquierda del botón.
  • data-bs-toggle="popover" data-bs-placement="top" muestra el popover en la parte superior del botón.
  • data-bs-toggle="popover" data-bs-placement="bottom" muestra el popover en la parte inferior del botón.

Tooltip and Popover - Ejemplo de la Prueba

1)"Docente":

  • data-bs-toggle="tooltip": marca el elemento para que Bootstrap le aplique tooltip.
  • data-bs-placement="bottom": coloca el tooltip debajo del elemento.
  • title="Información del docente": texto que muestra el tooltip.

Código:

<a class="nav-link" href="#" data-bs-toggle="tooltip" data-bs-placement="bottom" title="Información del docente">Docente</a>

2) "Comandante":

Código:

<a class="nav-link" href="Comandante.html" data-bs-toggle="tooltip" data-bs-placement="bottom" title="Abrir página Comandante">Comandante</a>

Imagen: img/Comandante.png
3) Botón "Buscar"

Botón para buscar en el sitio. Tooltip abajo.

Código:

<button class="btn btn-outline-light" type="submit" data-bs-toggle="tooltip" data-bs-placement="bottom" title="Buscar en el sitio">Buscar</button>
4) Botón lateral "LOGIN"

Tooltip a la derecha que indica la navegación a la página de inicio de sesión.

Código:

<a href="login.html" class="btn btn-secondary side-btn" data-bs-toggle="tooltip" data-bs-placement="right" title="Ir a la página de inicio de sesión">LOGIN</a>
5) Botón lateral "Registro"

Tooltip a la derecha que describe la acción de registrar un nuevo usuario.

Código:

<a href="formRegistro.html" class="btn btn-secondary side-btn" data-bs-toggle="tooltip" data-bs-placement="right" title="Registro de nuevo usuario">Registro</a>
6) Botón lateral "Producto"

Tooltip a la derecha que explica el propósito de crear o administrar productos.

Código:

<a href="formProducto.html" class="btn btn-secondary side-btn" data-bs-toggle="tooltip" data-bs-placement="right" title="Crear o administrar productos">Producto</a>
7) Botón para abrir el modal "Formulario"

Este proyecto inicializa también elementos con data-bs-tooltip="true".

Código:

Ejemplos de Popover con Bootstrap 5


Imagen: img/grupo_popover.png

10) "Grupo 2":

  • data-bs-toggle="popover": activa el popover.
  • data-bs-trigger="click": se abre al clic (no al hover).
  • data-bs-placement="right": aparece a la derecha del enlace.
  • data-bs-container="body": inserta el popover en <body> (evita problemas de overflow dentro del dropdown).
  • data-bs-flip="false": evita que cambie de lado automáticamente si no cabe.
  • title="...": título del popover (por ejemplo, "Grupo 2 - Integrantes").
  • data-bs-content="...": contenido del popover (texto o html breve).
<a class="dropdown-item" href="#" id="grupo2-link"
   data-bs-toggle="popover" data-bs-trigger="click"
   data-bs-placement="right" data-bs-container="body" data-bs-flip="false"
   title="Grupo 2 - Integrantes"
   data-bs-content="Muestra una tabla con los integrantes del Grupo 2 y sus correos.">Grupo 2</a>
