Progress Bar - Grupo 05

Teoría

Una barra de progreso es un componente visual utilizado para indicar el avance de una tarea o proceso en curso. Su propósito es ofrecer retroalimentación inmediata al usuario, mostrando de forma clara cuanto del proceso se ha completado.

En Bootstrap, las barras de progreso son totalmente adaptables, accesibles y personalizables. Gracias a su estructura modular, es posible modificar fácilmente su apariencia, color, animación y tamaño, manteniendo una presentación profesional y consistente. 

<div class="progress">
                          <div class="progress-bar" role="progressbar" aria-valuenow="70"
                              aria-valuemin="0" aria-valuemax="100" style="width:70%">
                              70% Complete
                          </div>
                        </div>
Características principales de las barras de progreso en Bootstrap:
  • Representan visualmente el progreso desde un valor mínimo hasta un máximo definido.
  • Son completamente responsivas y se adaptan al tamaño de su contenedor.
  • Permiten incluir animaciones dinámicas mediante la clase .progress-bar-animated.
  • Su color se puede personalizar con clases como .bg-success, .bg-warning o .bg-danger.
  • El porcentaje de avance se controla mediante style="width:X%".
  • Incorporan atributos ARIA para mejorar la accesibilidad (aria-valuenow, aria-valuemin, aria-valuemax).
  • Admiten texto interno, como el porcentaje o descripción del progreso.
  • Su altura puede modificarse fácilmente con CSS.
En conclusión, las barras de progreso son un componente fundamental en el diseño web moderno, especialmente útiles para representar el estado de procesos como descargas, cargas o instalaciones. Bootstrap ofrece una implementación flexible, moderna y accesible para este propósito.

1) Barras de progreso básicas

Las barras de progreso pueden mostrar diferentes porcentajes de avance mediante la propiedad width en línea.

Ejemplo
<div class="container">
                            <h1>Barras de Progreso</h1>
                            <div class="progress">
                              <div class="progress-bar" aria-valuenow="60" aria-valuemin="0" aria-valuemax="100" style="width:60%">
                                <span>60</span>
                              </div>
                            </div>
                          </div>

2) Barras de progreso contextual

Las barras de progreso contextual permiten transmitir distintos estados o niveles de avance mediante clases contextuales de Bootstrap.

  • .bg-success → Éxito o tarea completada.
  • .bg-info → Información o proceso en curso.
  • .bg-warning → Advertencia o precaución.
  • .bg-danger → Error o situación crítica.
Ejemplo (código HTML)
<div class="progress">
                        <div class="progress-bar progress-bar-successful" style="width:60%;">60%</div>
                      </div>

                      <div class="progress">
                        <div class="progress-bar progress-bar-info" style="width:60%;">60%</div>
                      </div>

                      <div class="progress">
                        <div class="progress-bar progress-bar-warning" style="width:60%;">60%</div>
                      </div>

                      <div class="progress">
                        <div class="progress-bar progress-bar-danger" style="width:60%;">60%</div>
                      </div>

5) Barras de progreso segmentada

Las barras de progreso también se pueden apilar dentro de un mismo contenedor para representar diferentes estados o categorías.

Crea una barra de progreso apilada colocando varias barras dentro del mismo <div class="progress">:

Ejemplo
<div class="progress">
                        <div class="progress-bar progress-bar-success" style="width:35%">
                          <span class="sr-only">35% completado (éxito)</span>
                        </div>
                        <div class="progress-bar progress-bar-warning" style="width:20%">
                          <span class="sr-only">20% completado (aviso)</span>
                        </div>
                        <div class="progress-bar progress-bar-danger" style="width:10%">
                          <span class="sr-only">10% completado (peligro)</span>
                        </div>
                      </div>

20% completado (aviso)

Ejercicio Práctico

A continuación se presenta un ejercicio práctico que ilustra como implementar una barra de progreso personalizada en una página web utilizando HTML, CSS y JavaScript.

En este ejercicio, se creará una barra de progreso que se llenará a medida que el usuario interactúe con un botón "Login". Se utilizarán estilos CSS para personalizar la apariencia de la barra y JavaScript para controlar su comportamiento.

Pasos a ser explicados de la integración del Progress Bar en el archivo de Prueba.
1. Estructura HTML con clases

En esta parte se puede ver el código de HTML implementado para la barra de progreso debajo de la etiqueta nav. 

<div id="barra-progreso-container" class="container text-center" style="display:none; margin-top:30px;">
                      <div class="progress" style="height: 30px; border-radius: 15px; background-color: #f0f0f0; box-shadow: 0 0 10px rgba(0,0,0,0.1);">

                          Cargando...

                      </div>
                    </div>

3. Estilos CSS Personalizados

Para dar una personalización adicional a la barra de progreso, se agregan estilos CSS personalizados que incluyen animaciones para mejorar la experiencia visual. 

#barra-progreso-container {
                      animation: aparecer 0.6s ease;
                    }

                    @keyframes aparecer {
                      from { opacity: 0; transform: translateY(-10px); }
                      to   { opacity: 1; transform: translateY(0); }
                    }

                    #barra-progreso::after {
                      content: "";
                      position: absolute;
                      top: 0;
                      left: 0;
                      height: 100%;
                      width: 100%;
                      background: linear-gradient(120deg, rgba(255,255,255,0.2), transparent);
                      animation: brillo 1.5s linear infinite;
                      border-radius: 15px;
                    }

                    @keyframes brillo {
                      0% { transform: translateX(-100%); }
                      100% { transform: translateX(100%); }
                    }
4. Evento y Animación con JavaScript

En un archivo JavaScript separado, se realiza la animación y el control de la barra de progreso al dar clic en el botón de "login" 

<script>
                    $(document).ready(function () {
                      $(".btn[href='']").first().on("click", function (e) {
                        e.preventDefault();

                        $("#barra-progreso-container").fadeIn();
                        let progreso = 0;
                        let intervalo = setInterval(function () {
                          progreso += 10;
                          $("#barra-progreso")
                            .css("width", progreso + "%")
                            .text("Cargando " + progreso + "%");
                          if (progreso >= 100) {
                            clearInterval(intervalo);
                            $("#barra-progreso-container").fadeOut();
                            $("#login").modal("show");
                          }
                        }, 200);
                      });
                    });
                    </script>
