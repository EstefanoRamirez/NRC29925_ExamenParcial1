Explorando jQuery

"Escribe menos, haz más": La biblioteca clásica de JavaScript.

¿Qué es jQuery?

jQuery es una biblioteca de JavaScript rápida, pequeña y rica en funciones. Facilita enormemente tareas como la manipulación y recorrido de documentos HTML (DOM), el manejo de eventos, la creación de animaciones y las peticiones Asíncronas (AJAX).

Su sintaxis fácil de usar la convirtió en la biblioteca de JavaScript más popular durante muchos años, simplificando la forma en que los desarrolladores interactuaban con la web.

Características Clave de jQuery
  • Selección de DOM: Potentes selectores para encontrar y manipular elementos HTML.
  • Manejo de Eventos: Un método simplificado para responder a las acciones del usuario (clics, teclado, etc.).
  • Efectos y Animaciones: Funciones integradas para crear animaciones personalizadas fácilmente.
  • AJAX: Simplifica el envío y recepción de datos del servidor sin recargar la página.
  • Compatibilidad: Abstrae las diferencias entre navegadores para un código más consistente.
Cómo Empezar con jQuery

Para empezar a usar jQuery, solo necesitas incluir la biblioteca en tu HTML, usualmente antes de tu propio script y al final del ``.

Tu proyecto ya incluye jQuery localmente (`vendor/jquery/jquery.min.js`), pero también puedes usar un CDN:

<script src="https://code.jquery.com/jquery-3.7.1.min.js"></script>

Un ejemplo básico para cambiar un texto:

$(document).ready(function(){
    $("h1").text("¡jQuery está funcionando!");
});
